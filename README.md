# Laravel ElasticSearch By Example

Example for working with Elastic Search in Laravel

System

- PHP 8.0
- MySQL 8.0
- Composer 2.0
- Elastic 7.x

## Elastic

Check Elastic status

```shell
sudo systemctl status elasticsearch.service
```
